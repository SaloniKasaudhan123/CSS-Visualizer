CSS Visualizer

CSS Visulizer is a showcase of the power of CSS. This project demonstrates how creative designs,animations and interactive elements can built without using Javascript.
The goal is to inspire designers and beginners to explore the visual potential of CSS and realize that styling can be as powerful as coding logic.
